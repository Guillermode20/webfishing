# WebFishing

A web-based fishing mmolite.

## Features

- Fishing
- Chatting

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS

### Backend
- Node.js
- Express
- WebSocket for real-time updates

### Development
- Vite

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
